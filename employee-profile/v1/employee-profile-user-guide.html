What is Employee Profile?

Overview

Employee Profile is the central record for all employee information in Bayzat. Access it via Company ‚Üí All Employees or use the global search (CMD/CTRL+K).

Each profile contains personal details, work information, documents, and leave balances. Admins can view up to 500 employees at once and search by name, role, or department.

Key Benefits

  • Single source of truth for employee data across HR, payroll, and attendance modules
  • Global search accessible from anywhere in the platform (CMD/CTRL+K)
  • View recently accessed profiles and search history for quick navigation
  • Configurable permissions control what employees can edit
  • Automated profile photo management and document tracking
  • Customizable employee directory fields for remote team collaboration

Product Foundation Overview

How Employee Profile Fits

Employee Profile is transactional data that stores individual employee information across personal, work, and document categories. It serves as the central record for all employee-related data in Bayzat.

Mental model: Employee Profile (record) ‚Üí Contains personal/work/document data ‚Üí Referenced by all HR modules ‚Üí Drives payroll, attendance, leave, and compliance processes

Each employee profile is unique and affects multiple modules. Changes to profile data (like hire date, salary, or work location) immediately impact leave balances, payroll calculations, and attendance tracking.

Key Decisions Before Setup

Answer these questions before creating employee profiles:

  • What profile data is mandatory vs. optional? ‚Äî Determine which fields are required for compliance and payroll processing
  • Who can edit employee information? ‚Äî Define whether employees can update their own data or if admin approval is required
  • How will you handle profile photos? ‚Äî Decide if admins will upload photos or if employees will add them themselves
  • What document types need tracking? ‚Äî Identify required employee documents with expiry dates for compliance

Related Features

  • Payroll ‚Äî Uses employee salary, bank details, and work information to calculate monthly compensation
  • Leave Management ‚Äî References hire date, work schedule, and assigned policies to calculate leave balances
  • Attendance ‚Äî Uses office location and work timings to track check-ins and calculate violations
  • Document Management ‚Äî Stores and tracks employee documents with expiry reminders

Initial Setup

First-Time Configuration

Before employees can access their profiles, complete these setup steps:

  1. Add company logo ‚Äî Navigate to Settings ‚Üí Company ‚Üí Company Logo ‚Üí Change logo. Upload your company logo (appears on platform and payslips).
  2. Configure office locations ‚Äî Go to Settings ‚Üí Company ‚Üí Offices ‚Üí Add New. Enter office name, location, and attendance check-in radius.
  3. Set up document types ‚Äî Access Settings ‚Üí Company ‚Üí Employee Document Types ‚Üí Add New. Define custom document categories with mandatory status and expiry tracking.
  4. Create employee records ‚Äî Navigate to Company ‚Üí All Employees. Add employees manually or via bulk upload with required fields (name, email, hire date, department, role).
  5. Send platform invitations ‚Äî From All Employees tab, click Send Invite next to each employee name. Employees receive email invitations valid for 3 days.

Key Tasks

Task: Search and Access Employee Records

Locate and view employee profiles using global search or list navigation.

Subtask: Use Global Search (CMD/CTRL+K)

Press CMD/CTRL+K from anywhere in the platform to open global search. View recently viewed employees (up to 5) and recent searches (up to 5) before typing. Type employee name to see matching results, then click to navigate to their profile.

Subtask: Browse Employee List

Navigate to Company ‚Üí Employees section. Select rows per page (10/25/50/100/250/500) to control display density. View employee list with name, role, and department columns. Click employee row to open their profile.

Task: View and Navigate Employee Profile

Access comprehensive employee information organized across multiple tabs.

Subtask: Access Profile Tabs

From employee profile, navigate between tabs: Personal (contact details, demographics), Work (job information, reporting structure), Documents (uploaded files), Leave (time-off balances), Attendance (work timings, check-in history), Payroll (salary components, payment details).

Subtask: View Profile Summary

Profile header displays employee photo (or initials circle), name, job title, department, and office location. Quick-access actions available in header for common tasks.

Task: Update Personal Information

Edit employee contact details, demographics, and personal data.

Subtask: Edit Contact Information

Navigate to employee profile ‚Üí Personal tab. Click Edit in Contact section. Update work email, personal email, phone numbers, and emergency contact. Save changes. Note: Work email field is greyed out if employee has already registered.

Subtask: Edit Demographics

In Personal tab, click Edit in Demographics section. Update nationality, date of birth (future dates disabled), gender, marital status, and passport details. For UAE employees, additional fields appear: Residence Visa Location, Trade License, MOL No. (shown only when Legal Country of Residence is UAE). Save changes.

FAQs & Support

Frequently Asked Questions

How do I delete an employee profile?

Navigate to Company > All Employees, search for the employee, select their checkbox, and click the Delete icon. This action is permanent and cannot be undone. If you want to keep the record but remove access, use offboarding instead.

Can I recover a deleted employee profile?

Profile recovery is limited. Profiles can be retrieved in some cases, but not all associated employee data can be restored. Contact support immediately if you accidentally delete a profile.

How do I resend an invite sent to the wrong email?

Delete the existing incorrect invite, update the employee's work email in their Personal tab, then resend the registration invite. This only works if the employee hasn't registered yet.

Why can't I edit an employee's work email?

If the employee has already registered, the work email field is locked. Contact support via the chat feature to request an email change.

How long are employee invitations valid?

Invitations expire after 3 days. Bayzat automatically sends 3 email reminders. If the employee doesn't register within this timeframe, you must resend the invite.

Can I upload profile photos for employees who haven't added them?

Yes, admins can upload or change employee profile pictures directly from the employee profile page on both web and mobile platforms.

How do I assign an office location to an employee?

Go to Company > All Employees, select the employee profile, navigate to the Work tab, click Edit, select the office location, and save. You can also use bulk upload for multiple employees.

Why is the avatar upload process slow?

The system processes image previews during upload, which can cause delays. Avoid clicking save multiple times while the image is processing.

Can I see full names in a single column when exporting employee data?

The platform currently splits names into separate columns (preferred name and last name). A unified full name field is not available in exports.

How do I view the organization chart?

Navigate to Company > All Employees (Super Admins) or People tab, then click the Organization chart view button. The chart only displays if at least one employee has the 'Reports to' field filled out.

Can I configure which fields employees can edit in their profiles?

Yes, admins can configure permissions to restrict employee editing of documents, personal/work info, and dependent data. This is useful for larger organizations to maintain data integrity.

How do I search for employees in large companies?

Use the global search accessible via CMD/CTRL+K from anywhere in the platform. You can view up to 500 employees per page, see recently viewed profiles, and access recent search terms.
