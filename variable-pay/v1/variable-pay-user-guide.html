What is variable-pay?

Overview

Variable Pay manages non-recurring compensation items like bonuses, commissions, allowances, and one-time payments. Access it through Settings ‚Üí Payroll ‚Üí Variable Pay Categories or directly from the payroll table.

Create custom pay categories, assign them to employees via the payroll table, and track all variable compensation in one place.

Key Benefits

  • Replace manual bonus tracking spreadsheets with system-managed records
  • Configure unlimited custom pay categories (taxable or non-taxable)
  • Bulk upload variable pay items across multiple employees
  • Automatic inclusion in payroll runs and WPS file generation
  • Complete audit trail of all variable compensation entries

Initial Setup

First-Time Configuration

Before using Variable Pay, configure pay item categories in Settings.

Create Variable Pay Categories

  1. Navigate to Settings ‚Üí Payroll ‚Üí Variable Pay Categories
  2. Click + Add Category
  3. Enter category name (e.g., Bonus, Commission, Overtime)
  4. Select calculation type (Fixed Amount or Percentage)
  5. Set tax treatment (Taxable or Non-Taxable)
  6. Click Save

Categories appear in the payroll table once created and can be assigned to employees.

Required Settings

Complete these configurations before processing payroll with Variable Pay:

  • Payroll Cycle: Ensure payroll cycle dates are set in Settings ‚Üí Payroll ‚Üí Payroll Cycle (contact support to modify cycle dates if needed)
  • Employee Records: Add employees to the Payroll Management table before assigning Variable Pay items
  • Category Configuration: Create at least one Variable Pay category (Recommended: create all categories your organization uses before entering data)

Access Verification

Confirm you can access Variable Pay features:

  1. Navigate to Payroll ‚Üí Payroll Table
  2. Click on any employee row to open the detail panel
  3. Verify Variable Pay tab appears in the employee detail view

If Variable Pay tab is missing, contact support to enable the module.

Key Tasks

Task: Manage Variable Pay Categories

Configure pay categories that define additional compensation items added to employee payroll.

Subtask: Create Variable Pay Category

  1. Navigate to Settings ‚Üí Payroll ‚Üí Variable Pays Categories
  2. Click "+ Add Variable Pay Category"
  3. Enter category name (e.g., "Overtime", "Commission", "Bonus")
  4. Add optional description
  5. Click Save

Expected Outcome: New category appears in the Variable Pays Categories list and becomes available for selection when adding pay items to employees.

Subtask: View Variable Pay Categories

Access Settings ‚Üí Payroll ‚Üí Variable Pays Categories to view all categories. Each row shows category name and description.

Subtask: Delete Variable Pay Category

  1. Locate the category in the Variable Pays Categories list
  2. Click the delete icon
  3. Confirm deletion in the popup

Expected Outcome: Category is removed from the system and no longer available for selection.

Task: Add Variable Pay to Employee Payroll

Add one-time or recurring variable compensation to individual employees.

Subtask: Add Variable Pay via Payroll Table

  1. Navigate to Payroll ‚Üí Payroll Table (see Payroll Management guide)
  2. Locate the employee row
  3. Click the More Options (three dots) menu
  4. Select "View Details"
  1. In the employee detail panel, locate the variable pay section
  2. Select the variable pay category from the dropdown
  3. Enter the amount
  4. Specify the effective date
  5. Click Add or Save

Expected Outcome: Variable pay item appears in the employee's payroll record and will be included in the next payroll run.

Subtask: Add Variable Pay via Adjustments Tab

  1. Navigate to Payroll ‚Üí Adjustments tab
  2. Click "+ Add Adjustment"
  3. Select employee
  4. Choose variable pay category
  5. Enter amount and effective date
  6. Click Save

Expected Outcome: Variable pay adjustment is recorded and reflected in the employee's payroll calculations.

Task: Bulk Upload Variable Pay

Subtask: Prepare and Upload Variable Pay File

  3. Download the template file
  4. Fill in employee identifiers, variable pay categories, amounts, and dates
  5. Upload the completed file
  6. Review validation results
  7. Confirm upload

Expected Outcome: Variable pay items are added to all specified employees in bulk.

Edge Cases

  • Variable pay with deleted leave requests: If a leave request is deleted after being processed, the payroll deduction remains but loses reference to the original request. Document leave details before deletion.
  • Inactive employees with arrears: Arrears persist in the system even after employee inactivation. Plan for manual cleanup or support intervention.
  • Payroll module reset: Resetting the payroll module requires complete data deletion and reconfiguration. Ensure backups exist before requesting a reset.
  • Work expenses in closed cycles: Expenses added to closed payroll cycles cannot be bulk-deleted. Process deletions before closing cycles or contact support.

Data Integrity Checks

FAQs & Support

Frequently Asked Questions

Can I bulk upload variable pay items for multiple employees?

Mass upload does not currently support rows with blank fields for specific pay item types. Each row must have complete data for the pay item type being uploaded.

How do I access pay item history when payroll cells are disabled?

Pay item history is not accessible when payroll table cells are disabled. You must enable the cell to view historical data.

Can I change the payroll cycle dates myself?

Payroll cycle date changes require backend intervention by the support team. Contact support to request a cycle date modification.

What happens to deleted employees in the payroll table?

Deleted employees are not automatically removed from the payroll table. Manual cleanup or support intervention is required to remove them.

Can I delete salary arrears for inactive employees?

Salary arrears cannot be deleted, even for inactive employees. Adjustments can be deleted, but arrears remain in the system.

How do I delete all work expense requests at once?

Bulk deletion of work expenses is not available in the UI. Contact support for backend deletion of historical expense records.

Can I reset the payroll module without support?

Payroll module reset requires support team intervention. There is no self-service option for resetting payroll data and configuration.

Are leave request details retained in payroll after deletion?

Leave request details are not retained in payroll deductions if the original leave request is deleted. Ensure leave requests are processed before deletion if payroll tracking is needed.
