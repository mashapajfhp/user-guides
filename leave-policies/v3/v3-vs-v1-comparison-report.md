1. **Dedicated Approval Flow section** with 3 request types (Leave, Leave Salary, Leave Encashment) and 4 routing examples
2. **5 navigation paths** with individual screenshots vs V1's single path
3. **Employee View subsection** (view balance, submit request, check status, edit pending, team calendar)
4. **Workflow Integration examples** (3 automation scenarios + 6-row comparison table + 3 future opportunities)
5. **Bayzat Insights badge** integration for leave history analysis
6. **25 unique screenshots** with descriptive captions (vs 13 in V1)
7. **13 cross-references** to related user guides (Leave Management, Attendance, Leave Salary, etc.)
8. **Policy creation wizard** documented as 4-step flow (Setup > Add Employees > Employee Details > Review & Save)
9. **Leave calendar** section with color coding (green=approved, yellow=pending) and public holiday integration
10. **Real test environment data** referenced (67 policies, 1,617 requests)
