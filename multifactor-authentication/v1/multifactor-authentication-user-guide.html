What is multifactor-authentication?

Overview

Multi-factor Authentication (MFA) adds a second security layer to your Bayzat login by requiring a time-based verification code from Google Authenticator. Access this feature through Profile > Security Settings to enable or disable MFA for your account.

Users can enable MFA through a 3-step setup wizard, scan a QR code (or enter a manual setup key), and verify with a 6-digit code that changes every 30 seconds.

Key Benefits

  • Protects against unauthorized access even if passwords are compromised
  • Required for payment-critical roles (Accounts Payable, Finance) to secure financial operations
  • Integrates with Finance module actions including card requests, expense approvals, and balance views
  • Email notifications confirm when MFA is disabled, alerting you to potential security changes

Product Foundation Overview

How Multi-factor Authentication Fits

Multi-factor Authentication is a user-level security control that adds a second verification layer to login processes. Each user manages their own MFA status independently through Profile settings.

Mental model: User enables MFA ‚Üí Google Authenticator generates codes ‚Üí User enters code at login ‚Üí Access granted

MFA affects individual user accounts only. Enabling or disabling MFA does not impact other users or system-wide settings.

Initial Setup

Prerequisites

Before enabling MFA, ensure you have:

  • A mobile device running iOS or Android
  • Google Authenticator app installed (download from App Store or Google Play)
  • Access to your Bayzat account profile settings

First-Time Activation

Navigate to Profile ‚Üí Security Settings to begin MFA setup.

Step 1: Enable MFA

  • Click Enable next to Multi-factor Authentication
  • Setup wizard dialog opens with 3-step process

Step 2: Link Authenticator App

  • Open Google Authenticator on your mobile device
  • Scan the QR code displayed on screen
  • If scanning fails, use the manual setup key provided

Step 3: Verify Setup

  • Enter the 6-digit code from Google Authenticator
  • Click Verify to complete activation
  • Confirmation message appears and status changes to Enabled with green indicator

Note: The authentication code changes every 30 seconds. Enter the current code displayed in your app.

Post-Setup Verification

After activation, verify MFA is working:

  • Log out of your Bayzat account
  • Log back in using your credentials
  • Enter the MFA code when prompted on the second authentication screen
  • Successful login confirms MFA is active

FAQs & Support

Frequently Asked Questions

How do I enable Multi-factor Authentication on my account?

Navigate to Profile ‚Üí Security Settings, click Enable for Multi-factor Authentication, download Google Authenticator app, scan the QR code (or enter the setup key manually), and enter the 6-digit verification code to confirm activation.

What happens if I lose access to my Google Authenticator app?

You must have access to Google Authenticator to disable MFA. If you've lost your device, contact Bayzat support immediately for manual backend assistance, especially if you have finance-critical roles like Accounts Payable.

How do I disable Multi-factor Authentication?

Go to Profile ‚Üí Security Settings, click Disable next to Multi-factor Authentication, then enter a verification code from Google Authenticator to confirm. You'll receive an email notification recommending re-enablement.

What should I do if my MFA code is rejected during login?

Wait for the next 6-digit code to generate (codes refresh every 30 seconds) and try again. Ensure your device time is synchronized correctly with network time.

Can I use MFA on both mobile app and desktop web?

Yes. Once enabled, MFA applies to all login methods. On mobile, enter the code after your credentials. On desktop (app.bayzat.com), enter the code on the second authentication screen.

Why can't I disable MFA for users with Accounts Payable roles?

The platform restricts MFA disablement for finance-critical roles to maintain payment security. Standard admin interfaces cannot override this setting‚Äîcontact Bayzat support for manual intervention.

Does MFA work with Corporate Cards and Expense modules?

Yes. MFA is integrated with Finance modules including Corporate Cards, Expense Overview, card creation/editing, card unfreezing, and balance viewing for enhanced payment security.

What if I can't scan the QR code during setup?

Use the manual setup key provided during the activation process. Enter this key directly into Google Authenticator instead of scanning the QR code.
