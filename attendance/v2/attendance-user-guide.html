What is Attendance?

Overview

Attendance tracks employee work hours through mobile check-ins, biometric devices, or manual entry. Access it from Time ‚Üí Employee Attendance.

The system records check-in/check-out times, calculates worked hours, and flags violations like late arrivals or absences.

Key Benefits

  • Eliminate manual timesheet collection and data entry
  • Reduce payroll errors from incorrect hour calculations
  • Enforce attendance policies automatically with configurable rules
  • Track overtime, deductions, and time-off impacts in one system
  • Support flexible work arrangements including shifts and remote work

Product Foundation Overview

How Attendance Fits

Attendance is transactional data that captures employee work hours through check-ins and check-outs. It sits at the operational layer, feeding into payroll calculations and time-off tracking.

Mental model: Work Timings (template) ‚Üí Assigned to Employees ‚Üí Check-in/Check-out Events ‚Üí Attendance Records ‚Üí Payroll Impact

One attendance record affects one employee for one day. Changes to work timings affect all future attendance calculations.

Initial Setup

Before employees can track attendance, configure these foundational settings. Complete setup in this order to ensure smooth operation.

1. Define Office Locations

Create office entries with geographical coordinates for location-based check-ins.

Settings ‚Üí Attendance ‚Üí Offices
  • Add office name and address
  • Set geographical coordinates for geo-fencing
  • Define check-in radius (distance employees can be from office to check in)

2. Configure Work Weeks

Specify which days are working days versus rest days for your organization.

Settings ‚Üí Attendance ‚Üí Work Weeks
  • Select working days (e.g., Sunday to Thursday)
  • Mark weekend days
  • Save work week template

3. Create Work Timings

Define shift schedules with start/end times and attendance rules.

Settings ‚Üí Attendance ‚Üí Work Timings
  • Set shift start and end times
  • Configure late arrival threshold (grace period before marking late)
  • Set early departure threshold
  • Define break duration (total break time to deduct from work hours)
  • Choose flexible hours option if applicable
  • Set "Absent after" timing (when to mark employee absent if no check-in)

Note: The default work timing cannot be deleted. You can edit it or create additional timing schedules.

4. Enable Image Capturing (Optional)

Activate photo verification for check-ins to prevent proxy attendance.

Settings ‚Üí Attendance ‚Üí Image Capturing
  • Toggle image capturing ON
  • System captures employee photo during each check-in/check-out

5. Assign Employees to Settings

Link employees to their office, work week, and work timing.

People ‚Üí Employees ‚Üí [Select Employee] ‚Üí Work Tab
  • Select employee's assigned office
  • Choose work week template
  • Assign work timing schedule
  • Save employee work settings

6. Configure Absence Management

Set rules for automatic absence marking and Time & Pay Adjustments.

  • System automatically marks employees absent at start of day
  • Absence is cleared when employee checks in
  • If no check-in by "Absent after" time, absence entry is created in Time & Pay Adjustments

Important: Employees must be assigned to an office, work week, and work timing before they can check in/out. Without these assignments, attendance tracking will not function.

Key Tasks

Task: Track Employee Work Hours

Record employee check-ins and check-outs to monitor attendance and calculate work hours.

Subtask: Check In to Work

Employees open the Bayzat mobile app and tap the check-in button when arriving at work. The system captures location and timestamp.

Subtask: Check Out from Work

Employees tap check-out when leaving. The system calculates total hours worked minus break time.

Subtask: Record Break Times

Employees check in and out for breaks. The system deducts break duration from total work hours.

Task: Configure Work Timings

Define shift schedules, break rules, and attendance policies for different employee groups.

Subtask: Create Work Timing Schedule

Navigate to Settings ‚Üí Attendance ‚Üí Work Timings. Click Add New. Enter schedule name, start time, end time, and break duration. Set late arrival and early departure thresholds. Save the configuration.

Subtask: Enable Flexible Hours

In Work Timing settings, toggle "All Hours Worked" to count all time between check-in and check-out as compensable hours, regardless of scheduled shift times.

Subtask: Configure Break Rules

Set maximum break duration and specify whether breaks are paid or unpaid. Define grace periods for late returns from breaks.

Task: Set Up Attendance Tracking

Configure offices, work centers, and employee assignments to enable attendance monitoring.

Subtask: Create Office Locations

Navigate to Settings ‚Üí Attendance ‚Üí Offices. Click Add New. Enter office name, address, and geographical coordinates. Set geo-fence radius for location-based check-ins. Save the office.

Subtask: Define Work Centers

Go to Settings ‚Üí Attendance ‚Üí Work Centers. Click Add New. Name the work center, assign schedulers, select office locations, and add work timings. Assign employees to the work center. Save configuration.

Subtask: Assign Employees to Work Settings

Navigate to employee profiles. In the Work section, select office location, work week, and work timing. Save changes to activate attendance tracking for the employee.

Task: Schedule Employee Shifts

Create and publish shift schedules for employees working variable hours or rotating shifts.

Subtask: Create Individual Shift

Navigate to Time ‚Üí Shift Scheduler. Select work center and date. Click on employee cell. Choose office, work timing, and add optional notes. Save as draft or publish immediately.

Subtask: Schedule Shifts in Bulk

Open Schedule Planner. Select shift type (working day or day off). Choose office and work timings. Click cells to assign shifts to multiple employees. Save and publish all shifts at once.

Subtask: Copy Repeating Shifts

Hover over copy icon next to employee name. Select date range (up to 90 days). Choose target employees. Optionally overwrite existing draft shifts. Apply to paste shifts across the selected period.

Subtask: Schedule Split Shifts

Create first shift for employee. Click on same employee's cell for a different time slot on the same day. Configure second shift with different office or timing. System validates no overlap. Save both shifts.

Task: Review Attendance Reports

Monitor employee attendance, identify violations, and generate reports for payroll processing.

Subtask: View Daily Attendance Report

Navigate to Time ‚Üí Employee Attendance. Select Daily Report. Choose date and apply filters (department, office, team). Review check-in/check-out times, late arrivals, early departures, and absences.

Subtask: Generate Custom Attendance Report

Select Custom Report. Choose date range. Apply employee filters. View report in browser or download as Excel. Review attendance status, work hours, and violation details.

Subtask: View Location Visits

Select Location Report. Choose date range. Click on location count to see detailed breakdown of time spent at each site. Review total locations visited and hours per location.

Task: Edit Attendance Records

Correct attendance data for missed check-ins, system errors, or employee requests.

Subtask: Override Attendance Times

Navigate to Daily Attendance Report. Click Edit button on employee row. Modify check-in time, check-out time, or location. Add comments explaining the change. Save the override.

Subtask: Create Attendance Record Manually

Go to Settings ‚Üí Attendance ‚Üí Biometric Data. Click Add Row. Select employee, enter check-in date and time, enter check-out date and time. Save to create manual attendance record.

Subtask: Mass Upload Attendance Data

Download Excel template from Biometric Data section. Fill in employee names, check-in dates/times, check-out dates/times. Upload completed file. System validates and imports attendance records.

Task: Process Attendance Regularization Requests

Review and approve employee requests to correct their attendance records.

Subtask: Review Regularization Ticket

Navigate to Requests ‚Üí Employee Tickets. Click on Pending tab. Select ticket to view requested check-in/check-out changes, employee comments, and attached evidence.

Subtask: Approve or Reject Request

Review ticket details. Click Accept to approve and automatically update attendance record, or click Reject to deny request. Add comments explaining decision. System notifies employee of outcome.

Task: Configure Attendance Policies

Set up overtime, days in lieu, and deduction policies to automate compensation calculations.

Subtask: Create Overtime Policy

Navigate to Settings ‚Üí Payroll ‚Üí Overtime Policy. Click Add New. Name the policy. Choose hourly rate type (Single Rate or Custom Rate). Configure rate multipliers for weekdays, weekends, and holidays. Select salary component (Basic Salary or Basic + Allowances). Assign employees to policy. Save configuration.

Subtask: Set Up Days in Lieu Policy

Go to Settings ‚Üí Attendance ‚Üí Days in Lieu Policy. Click Add New. Name the policy. Set hours-to-day conversion rate (default 8 hours = 1 day). Assign employees. Save to enable automatic leave accrual for extra hours worked.

Subtask: Configure Deduction Policy

Navigate to Settings ‚Üí Attendance ‚Üí Deduction Policy. Click Add New. Name the policy. Configure separate formulas for absent days, late arrivals, and early departures. Set rate multipliers and salary components. Assign employees. Save policy.

Subtask: Enable Line Manager Controls

Go to Settings ‚Üí Attendance ‚Üí Policies. Toggle "Allow line managers to control compensation impact" to ON. Line managers can now review and modify overtime and deduction entries before HR approval.

Task: Manage Time and Pay Adjustments

Review and approve attendance-related pay adjustments for overtime, deductions, and absences.

Subtask: Review Pending Adjustments

Navigate to Time ‚Üí Time and Pay Adjustments. Click Pending tab. Review entries showing extra hours, late arrivals, early departures, or absences. Check calculated amounts and formulas.

Subtask: Approve for Payroll

Select entries to approve. Click "Approve for Payroll" to move entries to Approved tab. Approved entries flow to payroll table for salary calculation. Once in payroll, entries cannot be edited.

Subtask: Approve for Time Off

For overtime entries with Days in Lieu policy, click "Approve for Time Off". System converts extra hours to leave days and adds to employee's time-off balance.

Subtask: Reject Adjustment Entry

Select entry to reject. Click Reject button. Add mandatory comment explaining rejection. Entry moves to Rejected tab and does not affect payroll or time off.

Subtask: Edit Deduction Amount

Go to Approved tab. Click on entry with "Assigned to Payroll" status. Click Edit button. Modify deduction amount. Add mandatory remarks explaining change. Save changes. Choose action: Reject, Move to Pending, Edit again, or Add to Payroll.

Task: Schedule Extra Hours in Advance

Assign overtime to employees before they work, with options for pay or time-off compensation.

Subtask: Create Extra Hours Request

Navigate to Time ‚Üí Time and Pay Adjustments. Click "Schedule Extra Hours" button. Select employee and date. Choose compensation type (payroll or time off). Enter number of extra hours. Add optional notes. Save request.

Subtask: Employee Accept or Reject

Employee receives push notification on mobile app. Opens request in Time & Pay Adjustments. Reviews scheduled extra hours, manager name, and comments. Clicks Accept or Reject. System updates request status.

Task: Enable Facial Recognition

Configure facial recognition for attendance check-ins to prevent proxy attendance.

Subtask: Configure Facial Recognition Settings

Navigate to Settings ‚Üí Attendance ‚Üí Facial Recognition. Toggle "Enable Facial Recognition" to ON. Set retry limit (1-5 attempts, default 3). Choose fallback behavior: capture image or block check-in. Select employees (all or custom list). Save configuration.

Subtask: Upload Reference Photos

System pulls primary passport photos from employee documents. For employees without photos, click Upload button. Select employee and upload clear face photo. Photo becomes reference for facial recognition comparison.

Subtask: Employee Check-In with Facial Recognition

Employee taps Check In on mobile app. First-time users accept consent prompt. Camera opens to scan face. AWS Rekognition runs CompareFaces and Liveness Detection. If verified, attendance is marked. If failed, employee can retry up to configured limit.

Task: Use Kiosk Mode for Bulk Check-Ins

Enable attendance marking for employees without smartphones using a shared device.

Subtask: Access Kiosk Mode

Log into Bayzat app on tablet or smartphone. Go to Manager view. Navigate to Attendance Daily Report. Click "Switch to Kiosk Mode". Select office location.

Subtask: Mark Employee Attendance

Search for employee by name. Select employee from list. Choose Check In or Check Out. If image capture is enabled, take employee photo. Confirm attendance. Repeat for next employee.

Task: Delete Shifts in Bulk

Subtask: Select Shifts to Delete

Navigate to Shift Scheduler. Click Delete icon. Select work center. Choose date range (up to 6 months in advance). Select employee names. Choose shift status (draft or published). Click Delete button.

Within 30 seconds of deletion, click Undo button to restore deleted shifts. After 30 seconds, shifts are permanently removed and must be recreated manually.

Task: Configure Attendance Regularization

Enable employees to submit tickets for correcting their attendance records.

Subtask: Add Ticket Type

Navigate to Settings ‚Üí Tickets. Select Attendance Regularization. Enter ticket name and description. Choose to apply to all employees or custom selection. Mark policy as active. Save configuration.

Go to Settings ‚Üí Approval Flow. Select Attendance Regularization ticket type. Assign approval roles (line manager, HR manager). Define approval sequence. Save flow to activate ticket routing.

FAQs & Support

Frequently Asked Questions

Can employees check in if they're on leave?

No. Employees on approved leave are exempt from check-in/check-out requirements and cannot mark attendance for those days.

What happens if an employee forgets to check out?

Hours won't be counted until checkout is recorded. Line managers can manually edit attendance records to add missing checkout times.

Can I schedule multiple shifts for one employee in a single day?

Yes. Split shifts allow up to 2 shifts per employee per day, with independent check-in/check-out tracking for each shift.

How do I correct an employee's attendance record?

Employees can submit an Attendance Regularization ticket to request corrections. Approvers review and accept/reject the request, which automatically updates the record if approved.

Can line managers see overtime amounts?

Only if the super admin enables the visibility toggle in Attendance Settings. Otherwise, line managers can only review and approve overtime requests without seeing calculated amounts.

What's the difference between overtime and days in lieu?

Overtime compensates extra hours with additional pay. Days in lieu converts extra hours into time off (default: 8 hours = 1 day).

Can I delete published shifts?

You can delete shifts up to 6 months in advance, but not for past dates or the current day if the employee has already checked in.

How do I track attendance for employees without smartphones?

Use Kiosk Mode on a shared device, or upload biometric data via Settings ‚Üí Attendance ‚Üí Biometric Data using the mass upload feature.

Can I copy shifts to multiple employees at once?

Yes. Use the Copy icon next to an employee's name, then paste to selected employees across a date range (up to 90 days).

What if facial recognition fails during check-in?

Employees can retry up to the configured limit (1-5 attempts, default 3). After that, the system either falls back to image capture or blocks check-in based on admin settings.

Can I modify a deduction amount after it's calculated?

Yes, in the Approved tab under Time and Pay Adjustments. Click the entry, edit the amount, add mandatory remarks, and save. Once added to payroll, editing is locked.

How do I exclude specific employees from attendance reports?

This feature is not currently available. Use filters to focus on specific departments, offices, or teams instead.
