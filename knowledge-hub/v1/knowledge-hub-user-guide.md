## What is Knowledge Hub?

### Overview

Knowledge Hub is Bayzat's centralized knowledge management platform that transforms how organizations store, organize, and access company information. It replaces scattered documents and inefficient information retrieval with an AI-powered, hierarchical content system.

### Key Benefits

- **AI-Powered Search:** Find information instantly using natural language queries with Bayzat AI chatbot
- **Organized Content Structure:** Create Spaces, Folders, and Pages with granular permission controls
- **Seamless Collaboration:** Share knowledge via direct links or Newsfeed with role-based access
- **Smart Activity Tracking:** Monitor content updates and engagement with real-time activity feeds

### Who Uses This Feature?

| User Role | What They Do | Value Proposition |
|----|----|----|
| **HR Administrators** | Create and maintain company policies, procedures, and resources | Publish policies once and ensure every employee has access to the latest version‚Äîeliminating outdated documents circulating via email |
| **Knowledge Hub Managers** | Organize content structure and manage permissions across spaces | Structure company knowledge logically with granular access controls‚Äîensuring sensitive information reaches only authorized audiences |
| **All Employees** | Search for information, access resources, and stay updated on company knowledge | Find answers instantly with powerful search across all company documentation‚Äîrather than asking colleagues or searching through emails |
| **Department Heads** | Share department-specific documentation with controlled access | Maintain team-specific resources with appropriate visibility‚Äîkeeping sensitive department information secure while enabling collaboration |

### Permission Levels

| Level | Description | Use Case |

<div id="business-rules-limitations" class="section section">

## Business Rules & Limitations

### Core Business Rules

| Rule | Description | Impact |
|----|----|----|
| Permission inheritance | Folder and page permissions inherit from parent Space | Set permissions at Space level for consistent access |
| Admin override | Super Admins and KH Managers have full access to all Spaces | Cannot restrict admin access to any content |
| File size limit | Maximum 15 MB per file upload | Large files must be compressed or split |
| Trash retention | Deleted items kept for 30 days | Recovery possible within retention period |

### System Constraints

- Cross-links between documents can only be created on web (viewable on mobile)
- AI chatbot requires content to be indexed before searchable
- Mobile app syncs content but has limited editing capabilities

## Troubleshooting & Edge Cases

### Common Issues

| Issue | Cause | Resolution |
|----|----|----|
| Cannot see Knowledge Hub | Feature not enabled or no permissions | Contact administrator to enable access |
| Cannot create Space | User lacks Editor role or higher | Request Editor permissions from admin |
| File upload fails | File exceeds 15 MB limit | Compress file or split into smaller parts |
| Content not in AI search | Recently added content not indexed | Wait for indexing to complete (usually minutes) |
| Cannot change admin permissions | Super Admins always have full access | This is by design and cannot be changed |

### Deleted Content Recovery

<div class="info-box">

**30-Day Trash Retention:** Deleted Spaces, folders, and pages are moved to Trash and kept for 30 days. Access Trash from the Knowledge Hub menu to restore accidentally deleted content.

## Support Resources

### Frequently Asked Questions

<div class="faq-accordion">

How do I find information in Knowledge Hub?

<div class="faq-answer">

Use the Bayzat AI chatbot for natural language search, browse Spaces directly, or check your Favorites and Recent files on the Overview page.

</div>

Can I share content with external users?

<div class="faq-answer">

No, Knowledge Hub content can only be shared with registered employees on your Bayzat platform. Use Copy Link for internal sharing.

</div>

How do I recover deleted content?

<div class="faq-answer">

Access the Trash from Knowledge Hub menu. Deleted items remain for 30 days and can be restored. After 30 days, content is permanently deleted.

</div>

Why can't I edit a Space?

<div class="faq-answer">

You need Editor permission for that Space. Contact the Space owner or a Knowledge Hub Manager to request edit access.

</div>
